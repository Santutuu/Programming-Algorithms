How it works

The inventory is sorted by quantity from smallest to largest. The backend uses a fast QuickSort-style algorithm to order items efficiently before returning them.

QuickSort behavior

This means the sorting process picks a pivot and partitions items, reducing the number of comparisons needed on average.
